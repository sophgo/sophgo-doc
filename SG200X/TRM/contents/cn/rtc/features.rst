特点
----

RTC 具有以下特点：

- 提供芯片系统复位信号 (System Reset)

- 提供 32,768 Hz 计数时钟 (误差 < ±1%)

- 提供 32-bit 秒钟计数器及硬件校准电路，可让秒钟精准度达到 5ppm 水平

- 支持报警时钟配置，并产生报警中断

- 提供 2KB 空间 SRAM, 可保存软件代码或暂存数据

- 支持电池低压检测，并产生中断

- 支持按键触发芯片从休眠唤醒

- 支持软件触发芯片休眠、复位或过热重启

- 支持 Watchdog 触发芯片系统复位

- 支持报警触发芯片从休眠唤醒

- 上下电时序、复位时间间隔可配置

- 提供 1 个超低功耗模拟秒钟计数器 (32 bit to count from 1970~2106 year)
