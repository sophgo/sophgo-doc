IRRX 红外接口
-------------

概述
~~~~

通过 IRRX 单元接收外部红外数据。

特点
~~~~

1. 支持 NEC 编码模式 (含repeat code)

2. 支持 Philips RC5/RC6 编码模式

3. 支持 Sony 编码模式

4. 支持红外线唤醒功能

工作方式
~~~~~~~~

.. _diagram_irrx_work_method:
.. figure:: ../../../../media/image142.png
	:align: center

	IRRX 工作方式

软件预先设定接收的红外线资料格式, 当 IRRX 模块接收到红外线通讯信号后将其解码,符合预定格式的编码通过中断传送給 CPU, CPU 再依据编码进行相应操作。
