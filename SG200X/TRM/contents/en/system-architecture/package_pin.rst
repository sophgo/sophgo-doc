Packaging and Pin
-----------------

Packaging
~~~~~~~~~

.. only:: sg2002

	Using QFN package, the package size is 9mm x 9mm x 0.9mm. Pin spacing is 0.35mm. The total number of pins is 88. Please refer to the figure below for detailed package dimensions.
